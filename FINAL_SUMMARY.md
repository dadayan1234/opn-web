## 🚀 Cara Mengubah Backend URL (SUPER SIMPLE):

### Edit File `.env` Langsung:

1. Buka file `.env`
2. Ubah 2 baris ini:
   ```env
   NEXT_PUBLIC_API_BASE_URL=https://backend-baru.com/api/v1
   NEXT_PUBLIC_BACKEND_URL=https://backend-baru.com
   ```
3. Save file
4. Restart server: `npm run dev`
5. **SELESAI! ✅**
